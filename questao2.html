2ª Questão

A questão número 2 da lista pedia a elaboração de um procedimento que obtivesse o conjunto de vértices adjacentes a um determinado vértice i presente em G. Tal procedimento deveria ser implementado usando matriz e lista de adjacência, além de demonstrar o log de execução.

Comando da Questão


Dado o grafo não direcionado G abaixo:



2) Elabore um procedimento que obtenha o conjunto de vértices adjacentes a um determinado vértice i presente em G. Esse procedimento deve ser implementado usando matriz e lista de adjacência. Apresente log de execução.



Solução

O procedimento que obtém o conjunto de vértices adjacentes de um determinado vértice segue abaixo:


  • Matriz
  • Lista de Adjacência

Procedimento

GrafoMatriz.prototype.getVerticesAdjacentes = function(vertice) {
    if (this.matriz[vertice]) {
        var adjacentes = [];

        for (var i = 0; i < this.matriz.length; i++) {
            if (this.matriz[vertice][i] != 0 && i != vertice)
                adjacentes.push(i);
        }

        return adjacentes;
    }
};
A função para obter os vértices adjacentes de um determinado vértice dado é bem simples. A primeira linha apenas faz a verificação de que o vértice dado como entrada realmente existe no grafo e após essa confirmação, para cada valor que não for 0 na linha do vértice na matriz adiciona-se esse vértice à lista de adjacentes. Neste caso, ignoramos caso o vértice possua uma aresta direcionada para si mesmo. 
GrafoListaAdjacencia.prototype.getVerticesAdjacentes = function (vertice) {
    if (this.lista[vertice]) {
        return this.lista[vertice];
    }
};
Como já era esperado, a função de retorno dos vértices adjacentes com implementação de lista de adjacência é bem direta e simples e obviamente dispensa comentários.
