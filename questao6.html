6ª Questão

A 6ª questão da lista tinha como comando mostrar como funcionava a busca em profundidade para o dígrafo D, ou seja, informar a ordem de visita dos seus vértices. Além disso, o problema requisitava a elaboração de um procedimento computacional que validasse o resultado encontrado.

Comando da Questão


Dado o grafo não direcionado G abaixo:



6) Mostre como a busca em profundidade funciona para o dígrafo D, ou seja, informe a ordem de visita dos vértices. Em seguida, elabore um procedimento computacional para validar o resultado encontrado. Apresente o log de execução



Solução

A busca em profundidade partindo do vértice 0 retornará a ordem de visita 0 --> 2 --> 5 --> 6 --> 3 --> 7 --> 9 --> 8 --> 1 --> 4. Partindo do vértice 0, temos a opção de ir tanto para o vértice 2 quanto para o vértice 3. Para o nosso caso, escolhemos sempre ir para o vértice de menor índice, neste caso, vértice 2. A partir do vértice 2, encontramos o 5, depois o 6, deparando-se então com uma aresta de retorno para 2.

A busca então retorna do 6 para 5, do 5 para o 2, do 2 para o 0, partindo então para o vértice 3. A partir do vértice 3, temos 2 possibilidades, ir para o vértice 7 ou 8. Seguindo o padrão descrito acima, vamos para o vértice 7, em seguida para o 9, que possui uma aresta de retorno para 7. Retornamos então o caminho da busca, voltando para o vértice 7, em seguida para o vértice 3. Neste momento, nossa busca se encontra em 0 --> 2 --> 5 --> 6 --> 3 --> 7 --> 9. Partindo para o vértice 8 a partir do vértice 3, não encontramos nenhum adjacente de 8 que ainda não tenha sido visitado. A busca então retorna todo seu caminho até voltar para o vértice 0, tendo seu término em 16.

A busca é então reiniciada a partir do vértice que ainda não foi visitado, isto é, o vértice 1, que visita o 4, finalizando então a busca em profundidade do dígrafo da questão. Abaixo, temos o procedimento computacional para validar o resultado encontrado, e ainda oferecemos a opção de partir de qualquer vértice presente no dígrafo.

var dfs = function (graph, verticeInicial, listaVertices) {
    cor = {};
    predecessor = {};
    d = {};
    f = {};

    for (var u in listaVertices) {
        cor[listaVertices[u]] = 'BRANCO';
        predecessor[listaVertices[u]] = null;
    }

    tempo = 0;

    for (var u in listaVertices) {
        if (cor[listaVertices[u]] == 'BRANCO') {
            visitaDFS(listaVertices[u], graph);
        }
    }
};
var visitaDFS = function (u, graph) {
    cor[u] = 'CINZA';
    tempo++;
    d[u] = tempo;

    var adjacentes = graph.getVerticesAdjacentes(u);

    for (var i in adjacentes) {
        var v = adjacentes[i];

        if (cor[v] == 'BRANCO') {
            predecessor[v] = u;
            visitaDFS(v, graph);
        }
    }

    cor[u] = 'PRETO';
    tempo++;
    f[u] = tempo;
};
O procedimento DFS inicializa todos os vértices presentes no grafo com cor branca e predecessor nulo. A partir desta inicialização, para cada vértice não visitado do grafo, é chamado o procedimento visitaDFS. A função visitaDFS coleta os vértices adjacentes do vértice passado como parâmetro e visita cada vértice adjacente, caso o mesmo ainda não tenha sido visitado. Por fim, a função seta a cor do vértice como visitado e define seu tempo de término.
